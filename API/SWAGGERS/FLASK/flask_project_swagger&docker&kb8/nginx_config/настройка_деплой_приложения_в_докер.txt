1. установка докер
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
docker version

2. подготовка приложения пайтон
в терминале переходим в директорию проекта и запускаем приложение (python3 {name.py})
далее прописываем в докер файл нужные нам настройки, переходим в папку с проектом в терминаел и выполняем команду sudo docker build .
Далее выполняем sudo docker images и видим наш созданный образ
Пример докер файла:
FROM ubuntu

RUN apt update

RUN   apt install python3 python3-pip -y

RUN pip3 install flask
RUN pip3 install flask_restful
RUN pip3 install flask_swagger_ui

COPY . /FLASK_SWAGGER
CMD python3 /FLASK_SWAGGER/swagger_flask.py
3. Запуск контейнера

Выполняем команду sudo docker run -it -p 5000:5000 <image id> берем из команды sudo docker images

docker cp nginx-base:/etc/nginx/conf.d/default.conf ~/Загрузки - копирование конфига из контейнера
sudo docker cp ~/Загрузки/default.conf nginx-base:/etc/nginx/conf.d/ - копирование конфига в контейнер
sudo docker exec nginx-base nginx -t - проверка конфига
